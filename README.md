# BimTools

A simple BIM system for Rhinoceros that lets you create and manage building elements (like walls) with structured data.

### Requirements

- Rhinoceros 8

## Current Features

- Create simple wall elements in Rhino
- Each element gets a unique ID
- Store metadata on geometry

## Planned Features

- Support for more element types (doors, windows, etc.)
- Convert elements to Revit format
- More advanced tools
